QUICK RESTORATION STEPS:

1. EXTRACT FILES:
   tar -xzf project-files.tar.gz

2. INSTALL DEPENDENCIES:
   npm install

3. SETUP ENVIRONMENT:
   - Copy .env file and update with your Supabase credentials
   - Get credentials from: https://app.supabase.com/project/_/settings/api

4. APPLY DATABASE MIGRATIONS:
   - Run all SQL files in supabase/migrations/ folder in order
   - Or use complete_database_schema.sql for full setup

5. START DEVELOPMENT SERVER:
   npm run dev
